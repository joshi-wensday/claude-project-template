# Skill Idea: Weekly review

**Goal:** Review the last weeks addition to the notes repository (see the diffs), and creates a nice weekly summary report, and cleans up the note structure (moves things to archive etc)
**Trigger Phrases:** "weekly review", "do a review and cleanup" 
**Required Tools:** No
**Workflow Steps:**
1. Analyse the diffs
2. Create a weekly report following the template
3. Clean up the notes, move thigns out of inbox (with user validation), move finished items to archive, and move things to a sensible location given the flat PARA directory structure. Then push the weekly diffs with a standard commit message format. No branches needed.
