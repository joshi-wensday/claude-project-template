# Skill Idea: Documentation creator

**Goal:** Create a documentation for a given feature
**Trigger Phrases:** "Write documentation for X"
**Required Tools:** potentially knowledge of the docuvault repo
**Workflow Steps:**
1. Understand the feature
2. Understand the requirements for good documentation
3. Write it in a standard format

*Note: When ready to build, run "Use skill-creator to build a skill based on ideas/skills/[filename].md"*
