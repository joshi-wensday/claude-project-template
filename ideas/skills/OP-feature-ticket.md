# Skill Idea: Open project feature ticket creator

**Goal:** create a nice open project feature ticket corresponding to the PRD for a feature
**Trigger Phrases:** "Create a an open project feature ticket", "create and OP ticket"
**Required Tools:** no
**Workflow Steps:**
1. read the PRD
2. Write the relavent information (product spec, dev spec, in a standardised format for a feature ticket)
